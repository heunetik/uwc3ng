UWC3NG Information

Ultimate Warcraft3 Next Generation (UWC3NG) is more or less a raceless, skill-based implementation of the Warcraft3 Frozen Throne plugin. Additional skills, items, etc are planned. This mod is a derived work based on Warcraft3 Frozen Thrones, developed by Geesu, visit www.wc3mods.net for more details.

This is how you bind a key:

1) Type ` to get to the console (key to the left of number 1)
2) Type bind - shopmenu (displays shopmenu when u press the minus key)
3) Type bind = shopmenu2 (displays shopmenu2 when you press the equals key)
4) Type bind p wcward (to use the serpant ward ability)

UWC3NG allows you to have up to 3 ultimate skills as opposed to just one from the original Warcraft3.

In the skills menu (/selectskill) ultimate skills will say (name (bind...)), for example for the Teleport ultimate it says (Teleport: (bind key wcteleport)).
Once you learn this skill you can use it by binding a key to the wcteleport command, done by typing: bind key wcteleport in the console.

Say /commands to see the list of UWC3NG spoken commands and say /help for the UWC3NG help menu.
