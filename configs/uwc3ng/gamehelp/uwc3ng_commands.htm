UWC3NG Command List
say /war3menu - UWC3NG main menu
say /shopmenu - Item menu for slot 1
say /shopmenu2 - Item menu for slot 2
say /charsheet - Your character summary
say /level - Displays your rank and level

say /help - Displays the UWC3NG help menu
say /skillsinfo - Displays skill details
say /itemsinfo - Displays item details

say /who - Displays a list of players on the server
say /whois <name> - Displays charsheet for the stated player
say /whoid <id> - Displays charsheet for the stated id

say /selectskill - Displays the skill selection menu
say /selectattrib - Displays the attributes selection menu
say /selectresist - Displays the resistances selection menu
say /resetskills - Resets your skills so you can retrain them
say /resetattribs - Resets your attributes so you can retrain them
say /resetresists - Resets your resistances so you can retrain them
say /deletexp - Deletes all of your XP and skill points

In console: xp_table - Displays the XP/Level table
